SVG Logo Example

Here's an SVG logo that you can use directly in your HTML. Copy the SVG code below:

Regular Logo (for header):

<svg width="180" height="60" viewBox="0 0 180 60" xmlns="http://www.w3.org/2000/svg">
    <rect width="180" height="60" rx="8" fill="white"/>
    <circle cx="30" cy="20" r="20" fill="#fef3c7" opacity="0.8"/>
    <circle cx="150" cy="40" r="15" fill="#fef3c7" opacity="0.8"/>
    <text x="50" y="35" font-family="Arial" font-size="20" font-weight="bold" fill="#d97706">Sweet Delights</text>
    <text x="30" y="35" font-family="Arial" font-size="24" fill="#d97706">üç∞</text>
</svg>

<svg width="180" height="60" viewBox="0 0 180 60" xmlns="http://www.w3.org/2000/svg">
    <rect width="180" height="60" rx="8" fill="#111827"/>
    <circle cx="30" cy="20" r="20" fill="#374151" opacity="0.8"/>
    <circle cx="150" cy="40" r="15" fill="#374151" opacity="0.8"/>
    <text x="50" y="35" font-family="Arial" font-size="20" font-weight="bold" fill="white">Sweet Delights</text>
    <text x="30" y="35" font-family="Arial" font-size="24" fill="white">üç∞</text>
</svg>

You can use these SVG logos directly in your HTML files by replacing the image tags with the SVG code.

For example, replace:

<img src="images/logo.png" alt="Sweet Delights Logo" class="h-12 object-contain">

With the SVG code above.

Alternatively, you can save these SVGs as files:

  1. Save the regular logo SVG code as logo.svg
  2. Save the white logo SVG code as logo-white.svg
  3. Place them in your images folder
  4. Update your HTML to reference these files: <img src="images/logo.svg" alt="Sweet Delights Logo" class="h-12 object-contain">
